grunt-rm
========

Grunt multi-task that removes specified files.

## Installation

```
npm install grunt-rm
```

For more information see grunt.js documentation:
https://github.com/cowboy/grunt/blob/master/docs/toc.md

## Config

```javascript
rm: {
  foo: 'js/*/**',
  bar: [
    'css/reset.css',
    'css/style.css'
  ]
}
```
